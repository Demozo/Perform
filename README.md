# Perform
Perform is a program made for use with the Performance skills in Final Fantasy 14.
Playing chords is sort of possible, the program automatically spaces simultaneously pressed notes by 7ms.

## Usage
Download a JAR from the releases page and run the JAR from the command line.
When ran without any arguments a help screen is displayed, listing all available commands.
